    Overview

    Bay of Assets provides a unified REST API that proxies requests to Anthropic (Claude), Google (Gemini), and OpenAI (GPT) through a single endpoint and a single API key. All responses conform to the OpenAI Chat Completions schema, making it a true drop-in replacement anywhere you currently use api.openai.com.

    Fully OpenAI-compatible. Point your existing code at https://api.bayofassets.com/v1, swap the key, and everything works — streaming, tool calling, system prompts, and all.

    All models use the OpenAI Chat Completions request format. The API handles translation to the native provider format internally — you never have to change your payload structure.

    Authentication

    Every request must include your API key as a Bearer token in the Authorization header.

    HTTP Header
    Authorization: Bearer your-api-key-here

    Keep your key secret. Never commit it to source code, expose it in client-side JavaScript, or share it publicly.

    Claude Code / Anthropic SDK — set ANTHROPIC_BASE_URL = "https://api.bayofassets.com/" (root).

    Rate Limits

    Rate limits are enforced per API key. A 429 Too Many Requests response is returned when exceeded.

    Limit TypeLimitations
    Requests / minute (RPM)70
    Monthly creditsPlan-based
    Concurrent requestsUnlimited

    Credits system: Each request deducts credits based on tokens used. Different models have different credit weights — Claude Opus costs more than Haiku, for example. Unused credits roll over to the next month. Check your usage in the dashboard.

    boaclaw (Quick Setup)

    The fastest way to set up Claude Code and Factory AI Droid with Bay of Assets. One command configures everything — env vars, Factory models, and backups.

    Prerequisites

    #WhatHow
    1Node.js 18+nodejs.org
    2BOA API keybayofassets.com
    3Factory AI (optional)See Factory AI section below

    Install Node.js (first-time users)

    Windows:

    powershell
    winget install OpenJS.NodeJS.LTS

    macOS:

    bash
    brew install node

    Linux (Ubuntu/Debian):

    bash
    curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt-get install -y nodejs

    Verify with: node -v (should show v18+)

    Run boaclaw

    bash
    npx boaclaw@latest

    It will prompt for your API key and configure everything in one shot. Or pass the key directly:

    bash
    npx boaclaw@latest <your-boa-api-key>

    Want the boaclaw command available globally?

    bash
    npm i -g boaclaw
boaclaw <your-boa-api-key>

    What it does

    ✔ Backs up your Factory config to ~/.boaclaw/backup.json

    ✔ Adds all 12 models to Factory AI Droid (if installed)

    ✔ Sets env vars system-wide — falls back to current user if no admin rights

    ✔ Works on Windows, macOS, and Linux

    Restore Original Settings

    bash
    boaclaw --restore

    Removes all env vars boaclaw set and restores your original Factory config from the backup.

    Uninstall

    bash
    boaclaw --restore
npm uninstall -g boaclaw

    Claude Code

    Use Claude Code as a CLI coding agent routed through Bay of Assets. No Anthropic account needed.

    Easiest method: Run npx boaclaw@latest — it sets up Claude Code automatically. The manual steps below are for reference.

    1. Install Claude Code

    bash
    npm install -g @anthropic-ai/claude-code

    2. Set Environment Variables (manual)

    bash
    export ANTHROPIC_BASE_URL="https://api.bayofassets.com/"
export ANTHROPIC_AUTH_TOKEN="your-api-key-here"
export ANTHROPIC_DEFAULT_HAIKU_MODEL="claude-haiku-4-5-20251001"
export ANTHROPIC_DEFAULT_SONNET_MODEL="claude-sonnet-4-6-thinking"
export ANTHROPIC_DEFAULT_OPUS_MODEL="claude-opus-4-6-thinking"

    Important: After setting env vars, restart your terminal.

    • • New terminal — picks up automatically
    • • Current terminal — run source ~/.zshrc or source ~/.bashrc
    • • VS Code / GUI apps — restart the app or log out & back in

    Factory AI Droid

    If you also want custom models inside Factory AI Droid, install Factory first then run boaclaw — it auto-detects and injects all 12 models.

    boaclaw handles this automatically. Just run npx boaclaw@latest after installing Factory — it detects ~/.factory/settings.json and configures everything.

    macOS / Linux:

    bash

    Windows (PowerShell):

    powershell
    irm https://app.factory.ai/cli/windows | iex

    Go to app.factory.ai and create a free account. Then run droid once to generate ~/.factory/settings.json.

    3. Run boaclaw

    bash
    npx boaclaw@latest

    boaclaw detects the Factory config and injects all 12 models automatically. Safe to re-run.

    OpenCode

    Use OpenCode as a terminal AI coding assistant routed through Bay of Assets.

    1. Install OpenCode

    bash
    npm install -g opencode-ai

    2. Create the Config File

    Edit or create ~/.config/opencode/opencode.json:

    json
    {
  "$schema": "https://opencode.ai/config.json",
  "provider": {
    "openai": {
      "options": {
        "baseURL": "https://api.bayofassets.com/v1",
        "apiKey": "your-api-key-here"
      }
    }
  },
  "model": "openai/gpt-5.3-codex"
}

    3. Launch OpenCode

    bash
    opencode

    Switch models with /model inside the session — all GPT, Gemini, and Claude models appear under the openai provider.

    Chat Completions

    POST/v1/chat/completions

    Request Body

    ParameterTypeRequiredDescription
    modelstringrequiredModel ID (e.g. gpt-5.3-codex)
    messagesarrayrequiredArray of message objects with role and content
    streambooleanoptionalStream tokens via SSE. Default: false
    max_tokensintegeroptionalMax output tokens
    temperaturenumberoptionalSampling temperature 0–2. Default: 1
    toolsarrayoptionalTool definitions for function calling

    Example Request

    json
    {
  "model": "claude-sonnet-4-6-thinking",
  "messages": [
    { "role": "system", "content": "You are a helpful assistant." },
    { "role": "user", "content": "Explain quantum entanglement simply." }
  ],
  "stream": false,
  "max_tokens": 1024
}

    Response Format

    json
    {
  "id": "chatcmpl-xyz123",
  "object": "chat.completion",
  "created": 1750000000,
  "model": "claude-sonnet-4-6-thinking",
  "choices": [
    {
      "index": 0,
      "message": {
        "role": "assistant",
        "content": "Quantum entanglement is when two particles..."
      },
      "finish_reason": "stop"
    }
  ],
  "usage": {
    "prompt_tokens": 32,
    "completion_tokens": 118,
    "total_tokens": 150
  }
}

    Parameters Reference

    ParameterTypeDefaultNotes
    temperaturefloat1.00 = deterministic, 2 = very random
    max_tokensintmodel maxHard cap on output length
    streamboolfalseSSE streaming
    top_pfloat1.0Nucleus sampling probability cutoff
    stopstring[]—Stop sequences (up to 4)
    presence_penaltyfloat0GPT models only
    frequency_penaltyfloat0GPT models only
    toolsarray—Function calling tool definitions
    tool_choicestring/object"auto"Tool selection mode

    Basic Usage

    python
    from openai import OpenAI

client = OpenAI(
    api_key="your-api-key-here",
    base_url="https://api.bayofassets.com/v1",
)

response = client.chat.completions.create(
    model="claude-sonnet-4-6-thinking",
    messages=[
        {"role": "user", "content": "Summarise the last decade of AI progress."}
    ],
    max_tokens=1024,
)

print(response.choices[0].message.content)

    Streaming

    python
    with client.chat.completions.stream(
    model="gpt-5.3-codex",
    messages=[{"role": "user", "content": "Write a sorting algorithm."}],
) as stream:
    for text in stream.text_stream:
        print(text, end="", flush=True)

    Error Codes

    All errors follow the OpenAI error response format.

    HTTP StatusMeaningCommon Cause
    400Bad RequestMissing required field, invalid model name
    401UnauthorizedMissing or invalid API key
    402Payment RequiredMonthly credits exhausted
    429Too Many RequestsRate limit exceeded — wait and retry
    500Internal Server ErrorUpstream provider error; retry with backoff
    503Service UnavailableProvider temporarily unreachable

    Error Response Shape

    json
    {
  "error": {
    "message": "Rate limit exceeded. Please retry after 60 seconds.",
    "type": "rate_limit_error",
    "code": "rate_limit_exceeded"
  }
}
